
# AY Challenge

It's a room management challenge




## Run Locally

Clone the project

```bash
  git clone https://github.com/900bce/ay_challenge.git
```

Go to the project directory

```bash
  cd ay_challenge
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
